# Security Analyst Portfolio

A professional portfolio website for a Security Analyst, built with HTML, CSS, and JavaScript.
Featuring a "bluish-black" dark theme, editorial project layouts, and responsive design.

## Features
- **Home**: Dynamic "Cyber Grid" hero section and professional bio.
- **Projects**: Editorial-style list with custom abstract visualizations.
- **Skills**: Categorized technical skills for easy scanning.
- **Education & Experience**: Timelines of academic and professional history.

## Setup
To view the site locally, simply open `index.html` in your web browser.

## Deployment
This static site can be deployed easily to GitHub Pages, Netlify, or Vercel.
